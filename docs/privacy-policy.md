# Privacy Policy

Last updated: 2026-03-23

Tabs Manager is designed to organize tabs in the current Chrome window.

## Data Collection

Tabs Manager does not collect, transmit, sell, or share personal data.

## Permissions Used

### tabs

Used to read tab URLs in the current window and move tabs so pages from the same website are placed together.

### storage

Used only to save the user's language preference for the extension interface.

## Data Storage

The extension stores the selected UI language using Chrome storage. This data remains inside the user's browser environment and is not sent to any external server.

## Third-Party Services

Tabs Manager does not use analytics, ads, remote code, or third-party tracking services.
