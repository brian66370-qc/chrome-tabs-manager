# Chrome Web Store Listing

## Product Name

Tabs Manager

## Summary

Organize Chrome tabs by website and keep related pages together.

## Description

Tabs Manager helps you clean up busy Chrome windows by moving tabs from the same website next to each other.

If you often work with many tabs at once, this extension gives you a faster way to scan your window and find related pages without manually dragging tabs around.

Key features:

- Sort tabs by website domain in the current window
- Keep tabs from the same site grouped together
- Preserve the original order inside each group
- Follow the Chrome interface language on first launch
- Let users switch between English, Traditional Chinese, and Japanese
- Save language preference with Chrome sync storage

Good for:

- Research sessions with many open sources
- Watching several YouTube tabs at once
- Managing multiple docs, dashboards, or GitHub pages
- Keeping a cleaner window without changing your browsing flow
